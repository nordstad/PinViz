# Examples

Explore example diagrams and configurations.

## Built-in Examples

Use the CLI to generate built-in examples:

```bash
pinviz example bh1750 -o bh1750.svg
pinviz example ir_led -o ir_led.svg
pinviz example i2c_spi -o i2c_spi.svg
```
